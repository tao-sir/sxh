参赛规则
资格要求

1.须持有有效身份证明的中华人民共和国公民。

2.每人只能参加一支队伍,每支队伍需要5名选手，少于5人的队伍不能参赛。

3.比赛仅支持《王者荣耀》微信区用户参与。

4.本次比赛不接受个人报名参加，未成年须在家长陪同下参加。

赛事规则

1. 参赛选手须在主办方指定服务器环境下进行比赛；

2. 所有比赛均为5v5王者峡谷征召模式；

3. 比赛胜负由系统判定胜负为准；

4. 本次比赛最多接收512支队伍报名，初赛（512支战队）至复赛（256支战队）采用BO1（一局定胜负）赛制，决赛（16支战队）采用BO3（三局两胜）赛制；

5. 抽签、开设建房、进入房间须有效率，裁判发起命令3分钟内未作为的队伍视为弃权；

6. 比赛禁选开始至比赛结束全过程不得人为退出游戏，违者视为弃权；

7. 当参赛选手发生以下不公平游戏行为，包括但不限于：串通比赛，事先分割奖金，利用漏洞，窥屏，代打，作弊，干扰、辱骂、殴打对手，侮辱行为（包括对对手、主办方、现场观众等），演播干扰，非法犯罪行为等。现场工作人员有权对选手及其战队进行处罚。处罚包括但不限于：口头警告、罚款及/或没收奖金、判定弃权、禁赛、取消参赛资格等。

应急处置

1. 比赛开始后，一旦发生掉线、账号异常、设备异常、服务器崩溃等无法进行游戏的状况，选手需立即告知裁判，裁判暂停游戏。异常结束、所有选手均可继续正常游戏后，裁判告知双方后，由裁判结束暂停，比赛继续。

2. 如果选手在未得到裁判许可的情况下暂停或者取消暂停游戏，会被认为是影响比赛公平性的行为，王者荣耀赛事组可以进行仲裁。

3. 如果游戏无法暂停且无法正常继续进行，赛事组可裁定本局需要重新开始时，比赛双方需直接使用普通房间, 选择同样的英雄、召唤师技能、铭文等进入游戏，如有私自更换，裁判可直接判负。

其他说明

1. 本次赛事最终解释权在法律允许范围内归主办方所有。
